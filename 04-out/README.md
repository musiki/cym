# Bienvenidx

La definición de topoi como lugares comunes y dinámicos de conocimiento implica una acción performática en la organización de información mas que una actitud enciclopedísta. La tópica está definida como axiomas y vínculos entre los procedimientos artísticos y científicos. Por tanto, la información aquí presentada tiene como función generar procedimientos estéticos subjetivos, alejados de cualquier comprobación o pretensión científica.&#x20;
